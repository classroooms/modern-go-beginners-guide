# String Formatting

| Sr No. | Particular | Data Type                  |
| ------ | ---------- | -------------------------- |
| 1.     | %d         | integer                    |
| 2.     | %f         | float                      |
| 3.     | %s         | string                     |
| 4.     | %c         | character                  |
| 5.     | %q         | quoted character or string |
| 6.     | %v         | default format             |
| 7.     | %b         | binary                     |
| 8.     | %x         | hexadecimal                |
| 9.     | %t         | boolean                    |
